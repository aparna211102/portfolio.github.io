What is online whiteboard?


Online whiteboards can be used for the same purpose as a real whiteboard, but without the limitations of a real whiteboard. Made during pandemic (in the year 2021) when our classes were going on in online mode and it is hard to interact with teachers only via PPt presentation.

Online whiteboards are software solutions that come along as apps or browser-based tools. They support teams to collaborate visually over distance in real-time.

Why you should have this online whiteboard?


  • Improve collaborative processes when working remotely.

  • Boost the efficiency of online meetings

  • The purpose of a whiteboard is to visualize thoughts, concepts, write down ideas, explain and teach, plan and create in the group and many other things.

  • Efficient for online classes.

  • During pandemic it is hard to learn things only via ppts and voice lecture, it will help in the same manner as whiteboard present in our classrooms.


What do I need to know to create this white-board?

    You can make a cool portfolio website for yourself with HTML, CSS, and JavaScript.


  1. Create the structure with HTML. The first thing you have to learn, is HTML, which is the standard markup language for creating web pages.

  2. Style with CSS. The next step is to learn CSS, to set the layout of your web page with beautiful colors, fonts, and much more.

  3. Make it interactive with JavaScript. After studying HTML and CSS, you should learn JavaScript to create dynamic and interactive web pages for your users.
